# Story: Implement Verse Rating (1-5)

**Story Point Estimate:** 5
**Priority:** High

## Description
Parity with Legacy App: Users must be able to rate their memory of a verse on a scale of 1-5.

## Requirements
1.  UI: Display 5-star or 1-5 number selection interface when reviewing a verse.
2.  Logic: Capture the rating.
3.  API: Call the `ratePerformance` endpoint (see legacy `MemverseApi.java`).
4.  Feedback: Show success/failure of rating submission.
