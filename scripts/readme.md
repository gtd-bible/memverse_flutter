# Scripts Directory Index

This directory contains key utility and deployment scripts for the MiniMemverse project.

## Index

- **deploy_ios.sh** — Main iOS deployment script (builds IPA, runs dry-run, uploads)
- **build_ipa_env.sh** — Alternate/legacy build script using env variables

## Instructions
- See usage in each script by running with `--help`.
- Scripts are designed for easy copy-paste deployment and CI/CD automation.

For full devops, TestFlight, and Firebase instructions, see docs in project root:
- `devops_checklist.md` — Progress tracking for deployment tasks
- `testflight_open_beta.md` — How to set up an open beta via TestFlight
- `testflight_open_beta_faq.md` — FAQ for testers/devs
